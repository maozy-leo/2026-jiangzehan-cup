# 2026 江泽涵杯数学建模项目


本仓库收录2026年北京大学江泽涵杯建模比赛 B 题参赛材料，包括赛题与原始数据、最终报告、各问题的源代码、实验输出和可视化结果。

项目作者为：
- 组长：毛志远，北京大学数学科学学院
- 组员：朱沐恩，北京大学光华管理学院
- 组员：董禹辰，北京大学数学科学学院

## 仓库结构

| 路径 | 内容 |
| --- | --- |
| [`report.pdf`](report.pdf) | 最终报告 |
| [`B题/`](B%E9%A2%98/) | B 题题目与原始数据压缩包 |
| [`附件/`](%E9%99%84%E4%BB%B6/) | 数据处理、网络攻击/修复算法、实验输出与图表 |
| [`附件/README.md`](%E9%99%84%E4%BB%B6/README.md) | 各问题脚本、参数和复现流程的详细说明 |

`附件/` 中的主要内容按题号划分：

- `Problem1/`：城市路网构建、统计、度分布分析和可视化。
- `Problem2/`：随机攻击 Monte Carlo 基线。
- `Problem3:4/`：自适应攻击策略、批处理、攻击序列与动态/静态可视化。
- `Problem5/`：基于介数中心性的网络修复与加固实验。

> `Problem3:4` 和 `3:4visualize` 的目录名包含半角冒号。在 shell 中建议始终用引号包围完整路径。

## 运行环境

建议使用 Python 3.10 或更高版本：

```bash
python -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
```

部分地图绘制功能需要联网获取底图；交互式动图需要可用的 Matplotlib 图形后端。详细命令和参数见 [附件说明](%E9%99%84%E4%BB%B6/README.md)。

## 数据与结果说明

仓库保留了论文复现所需的城市边表、攻击序列、统计表和已生成图像。部分边表在不同问题的工作目录中重复保存，是为了保持原实验脚本可独立运行。
